# US COVID 19 Tracker

Dashboard view of COVID 19 virus in the United States
Data from <https://covidtracking.com/api/>

## Installation

- git clone this project
- cd into project folder
- npm run start

## Built With

- react
- material-ui (dashboard template)
- recharts
- moment
- clsx
